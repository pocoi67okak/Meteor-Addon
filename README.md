## ⚙️ Настройки

### 🔧 General — Основные

| Настройка | Тип | По умолчанию | Описание |
|---|---|---|---|
| **Radius** | `1–10` | `4` | Радиус сканирования блоков вокруг игрока |
| **Delay** | `1–20` тиков | `5` | Задержка между действиями. 1 — быстро, 20 — медленно |
| **Harvest** | Вкл/Выкл | ✅ Вкл | Автоматически собирать созревший урожай |
| **Replant** | Вкл/Выкл | ✅ Вкл | Засаживать семена на **все** пустые грядки в радиусе |
| **Break** | Вкл/Выкл | ❌ Выкл | Автоматически ломать сорняки (wildflowers, short grass, tall grass) |
| **Hoe** | Вкл/Выкл | ❌ Выкл | Автоматически вспахивать землю мотыгой из инвентаря |
| **Rotations** | Режим | `Always` | Когда поворачивать голову к цели |
| **Debug** | Вкл/Выкл | ❌ Выкл | Отправлять локальные отладочные сообщения в чат |

#### Режимы поворотов (Rotations)

| Режим | Описание |
|---|---|
| `Always` | Поворачивать голову при любом действии |
| `Harvest` | Поворачивать только при сборе урожая |
| `Replant` | Поворачивать только при посадке |
| `None` | Не поворачивать голову |

## 🔄 Система приоритетов

Каждый тик модуль проверяет действия **по приоритету**:

1. 🌾 **Harvest** — Сбор созревших культур
2. 🌿 **Break** — Уничтожение сорняков
3. ⛏️ **Hoe** — Вспахивание земли
4. 🌱 **Replant** — Посадка семян на пустые грядки

> За один «тик действия» выполняется только **одно** действие, после чего срабатывает задержка (Delay).

## 📥 Установка

1. Скачай последний `.jar` файл из раздела [Releases](https://github.com/pocoi67okak/Meteor-Addon/releases)
2. Помести его в папку `mods` вместе с Meteor Client
3. Запусти игру — модуль появится во вкладке **Player** → **auto-crop**
